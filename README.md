# TTWatchBox

**一款XTC手表玩机工具箱，聚集了各种功能。**

### 部署方法：

你需要拥有：
- 一台至少 `Windows 10` 的电脑（目前仅限 Windows 系统）
- `Python` 版本至少为 12

接下来请按如下步骤部署：
1. 克隆仓库：
```bash
git clone https://github.com/TTchen-Dev-Team/XTCToolkitPro.git
cd XTCToolkitPro
```
2. 创建并进入虚拟环境
```bash
python -m venv .venv
.venv\Scripts\Activate
```
3. 一键部署
```bash
run
```
